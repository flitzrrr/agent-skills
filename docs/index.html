How It Works

Skills are markdown files (SKILL.md) that teach AI assistants specialized capabilities. Drop them into your project and the assistant picks them up automatically.

1

Install

Run npx @flitzrrr/agent-skills install or clone the repo. Skills are symlinked to each platform's config directory.

2

Use

Ask your AI assistant to use a skill by name. For example: "Use the security-review skill to audit this PR."

3

Stay Updated

Run npx @flitzrrr/agent-skills update to pull the latest skills. New sources are discovered and added automatically every Monday.

Sources

All skills come from established, open-source repositories. Each source is security-scanned before integration.

Platform Support

Works out of the box with all major AI coding assistants. The CLI auto-detects installed platforms and sets up skills accordingly.

Quality Gates

Every source is automatically scanned before integration. Sources that fail any gate are rejected.

Star Threshold
Minimum 50 GitHub stars. Ensures the source has community validation and active usage.
Skill Count
Minimum 3 SKILL.md files. Prevents single-purpose or placeholder repositories from being added.
Content Ratio
Minimum 30% markdown files. Skills should be documentation-heavy, not code-heavy.
Security Scan
No obfuscated code (eval/exec), no hardcoded credentials, no excessive executables.
License Check
Must have a recognized open-source license. Each submodule retains its original license.
Weekly Refresh
Discovery runs every Monday. New candidates must pass all gates before automatic integration.
