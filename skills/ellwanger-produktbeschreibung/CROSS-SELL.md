# Cross-Sell Matrix — Ellwanger Katalog

Jede Beschreibung soll 1–2 ergänzende Produkte aus dem Ellwanger-Sortiment nennen. Die Empfehlungen basieren darauf, was Vereine und Betreiber typischerweise zusammen kaufen.

## Matrix

| Kategorie | Cross-Sell zu | Beispielformulierung |
|---|---|---|
| Tennisnetze | Tennisnetzpfosten, Mittelbänder, Spannseile | „In Kombination mit den passenden Ellwanger-Netzpfosten…" |
| Tennisnetzpfosten | Tennisnetze, Bodenhülsen | „Zusammen mit einem Ellwanger-Tennisnetz…" |
| Tennislinien | Spanngeräte, Linienstampfer, Linienkehrer | „Für die Verlegung empfehlen wir das Ellwanger-Spanngerät…" |
| Linienkehrer | Tennislinien, Linienstampfer, Abziehbesen | „Nach dem Kehren sorgt ein Ellwanger-Schleppnetz für eine ebene Oberfläche…" |

| Schleppnetze | Walzen, Abziehgeräte, Abziehbesen | „In Kombination mit einer Ellwanger-Walze…" |
| Abziehgeräte | Schleppnetze, Abziehbesen, Walzen | „Ergänzend dazu eignet sich das Ellwanger-Schleppnetz…" |
| Abziehbesen | Abziehgeräte, Schleppnetze | „Für die Grundpflege empfehlen wir zusätzlich ein Ellwanger-Schleppnetz…" |
